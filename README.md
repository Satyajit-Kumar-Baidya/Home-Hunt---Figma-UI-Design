# Home-Hunt---Figma-UI-Design
Home-Hunt is a housing search platform designed to help both bachelors and families easily find suitable rental homes. The design focuses on providing a simple and user-friendly interface where users can browse available houses, view important details, and quickly decide which property fits their needs.
